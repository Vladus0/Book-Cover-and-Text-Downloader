# Book Cover and Text Downloader

## Описание

Данная программа предназначена для скачивания обложек и текстов книг. Она позволяет пользователю вводить названия книг, после чего программа находит и загружает соответствующие обложки и тексты из доступных источников.

### Как установить

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей::
```
pip install -r requirements.txt
```

### Запуск программы

Чтобы запустить программу вам нужно написать:
```
python main.py
```

Программа загрузит обложку и текст книги и сохранит их в папки.

### Дополнительные параметры запуска:

Для того чтобы указать параметры запуска после `python main.py` напишите команду для указания параметра

1. Напишите -h или --help чтобы получить подсказки в использовании параметров
   ```
   python main.py --help
   ```

2. Команда для указания первой страницы, с которой начнется собирание данных о книгах. Для вызова -f или --first и значение
   ```
   python main.py -f=страница с которой начнется собирание данных
   ```
3. Команда для указания последней страницы, на которой закончится собирание данных о книгах. Для вызова -l или --last и значение
   ```
   python main.py -l=страница на которой закончится собирание данных
   ```

4. При вызове команды при запуске будет пропускаться скачивание картинок книг. Для вызова -si или --skip_imgs
   ```
   python main.py -si
   ```
5. При вызове команды при запуске будет пропускаться скачивание текстов книг. Для вызова -st или --skip_txt
   ```
   python main.py -st
   ```
